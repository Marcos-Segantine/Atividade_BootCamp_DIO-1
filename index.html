Tag p

Este elemento cria um parágrafo na página.

Por exemplo, este texto que voce está lendo neste exato momento está dentro de uma tag p.

Tag a

Insere uma link seja para um apaguna externa ou para a propria pagina mesmo.

Por exemplo, os indice no topo da página são link (elementos a) usados para navegar dentro da própria página.

Tag img

Esta tag insere uma imagem na página, ela possui um atributo obrigatorio que é o src onde é especificado o local onde a imagem está, seja na internet ou local, e tambem tem um outro atributo muito importante que é o alt é um atributo de asscibilidade onde caso a imagem não seja carregda ou encontrada o texto de alt será exibido.

Tag ul

A ul é utilizada para criar uma lista não ordenada onde dentro dela terão os li que são os itens da lista.

Tag ol

Ao contrário da tag ul a ol inseri na página uma lista ordenada com os mesmos li dentro.

Tag li

Cada li representa um item de uma lista seja ela ordena ou não.
